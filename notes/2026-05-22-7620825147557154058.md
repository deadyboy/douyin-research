## 交付检查清单
在交付 UI 代码之前，请验证这些项目：
### 视觉质量
- [ ] 未使用表格行写作为图标（使用 SVG 代替）
- [ ] 所有图标来自单一的图标库（Heroicons/Lucide）
- [ ] 品牌图标正确（从 Simple Icons 验证）
- [ ] 悬停状态不会导致布局移动
- [ ] 直接使用主题颜色（bg-primary），不用 var() 包装
### 交互
- [ ] 所有可点击元素都有 "cursor:pointer"
- [ ] 悬停状态提供清晰的视觉反馈
- [ ] 过度平滑（150-300ms）
- [ ] 焦点状态对键盘导航可见
### 浅色/深色模式
- [ ] 浅色模式文本具有足够的对比度（最少 4.5:1）
- [ ] 玻璃/透明元素在浅色模式下可见
- [ ] 边框在两种模式中都可见
- [ ] 支持浏览器内切换模式
### 布局
- [ ] 浮动元素与边框的间距适当
- [ ] 内容未被遮盖在固定导航栏后面
- [ ] 在 375px, 768px, 1024px, 1440px 处响应式
